# A3数据绘图功能测试说明

## 功能概述

本次实现为A3协议添加了以下功能：
1. 动态曲线数量和自定义Legend名称
2. 两种数据排列方式的解析（交错和分块）
3. 数据排列方式选择界面

## 数据排列方式说明

### 1. 交错排列 (Interleaved)
数据格式：`[x0, y0, z0, x1, y1, z1, x2, y2, z2, ...]`

示例：3个通道(x,y,z)，10个数据点，数据类型float(4字节)
- 总数据长度：3 × 10 × 4 = 120字节
- 数据排列：x0(0-3), y0(4-7), z0(8-11), x1(12-15), y1(16-19), z1(20-23), ...

### 2. 分块排列 (Blocked)
数据格式：`[x0, x1, x2, ..., y0, y1, y2, ..., z0, z1, z2, ...]`

示例：3个通道(x,y,z)，10个数据点，数据类型float(4字节)
- 总数据长度：3 × 10 × 4 = 120字节
- 数据排列：x0-x9(0-39), y0-y9(40-79), z0-z9(80-119)

## Legend命名规则

- 如果表格中"说明"列有内容，使用该内容作为曲线Legend
- 如果"说明"列为空，使用数字累加命名（1, 2, 3, ...）

## 测试案例

### 测试案例1：3通道交错排列数据

**配置：**
- 协议类型：A3
- 参数数量：3个
- 参数1：名称"数据X"，类型"float"，长度4，说明"X轴"
- 参数2：名称"数据Y"，类型"float"，长度4，说明"Y轴"
- 参数3：名称"数据Z"，类型"float"，长度4，说明"Z轴"
- 排列方式：交错

**测试数据生成（Python示例）：**
```python
import struct
import math

# 生成10个时间点的数据
point_count = 10
data = bytearray()

for i in range(point_count):
    t = i * 0.1
    # x: 正弦波
    x = 50 + 30 * math.sin(t * 2 * math.pi / 2)
    # y: 余弦波
    y = 50 + 30 * math.cos(t * 2 * math.pi / 3)
    # z: 线性增长
    z = 10 * i
    
    # 交错排列：x, y, z
    data.extend(struct.pack('<f', x))
    data.extend(struct.pack('<f', y))
    data.extend(struct.pack('<f', z))

# 保存到文件
with open('test_interleaved.bin', 'wb') as f:
    f.write(data)

print(f"生成交错排列测试数据，长度: {len(data)} 字节")
```

### 测试案例2：3通道分块排列数据

**配置：**
- 协议类型：A3
- 参数数量：3个
- 参数1：名称"温度"，类型"float"，长度4，说明""（空）
- 参数2：名称"压力"，类型"float"，长度4，说明""（空）
- 参数3：名称"流量"，类型"float"，长度4，说明""（空）
- 排列方式：分块

**预期Legend：** 1, 2, 3

**测试数据生成（Python示例）：**
```python
import struct
import math

# 生成10个时间点的数据
point_count = 10
data = bytearray()

# 先生成所有x数据
for i in range(point_count):
    t = i * 0.1
    x = 50 + 30 * math.sin(t * 2 * math.pi / 2)
    data.extend(struct.pack('<f', x))

# 再生所有y数据
for i in range(point_count):
    t = i * 0.1
    y = 50 + 30 * math.cos(t * 2 * math.pi / 3)
    data.extend(struct.pack('<f', y))

# 最后生成所有z数据
for i in range(point_count):
    z = 10 * i
    data.extend(struct.pack('<f', z))

# 保存到文件
with open('test_blocked.bin', 'wb') as f:
    f.write(data)

print(f"生成分块排列测试数据，长度: {len(data)} 字节")
```

### 测试案例3：2通道混合说明

- 参数1：名称"通道A"，类型"float"，长度4，说明"电压(V)"

## 测试步骤

1. 在协议配置页面配置A3协议参数
2. 进入命令操作页面
3. 选择数据排列方式（交错/分块）
4. 点击"绘图"按钮打开绘图窗口
5. 发送测试数据到设备
6. 观察曲线是否正确显示
7. 检查Legend名称是否正确

## 注意事项

- 确保所有参数的dataLength正确设置
- 确保payload总长度能被单个数据点总长度整除
- 数据类型必须正确匹配（float是4字节）
- 如果说明为空，Legend会自动使用数字
